

- pv

```bash
# pvcreate用于创建物理卷
pvcreate /dev/nvme

# pvdisplay、pvsca、pvs用于查看物理卷pvdisplay
pvspvscan
```



- vg

```bash
# 使用vgcreate创建卷组
vgcreate VG_NAME device1 ... devicen

# 使用vgdosplay、vgscan、vgs命令查看卷组
vgdisplay
vgscan
vgs
```



- lv

```bash
# 使用lvcreate创建逻辑卷

lvcreate -l 100%VG -n lvdata vgdata

使用lvdisplay、lvscan、lvs查看逻辑卷
lvdisplay
lvscan
lvs
```
